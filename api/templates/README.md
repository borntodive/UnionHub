# Template Carta Intestata

Carica qui il tuo template PDF della carta intestata CISL.

## File richiesto

- `letterhead.pdf` - Il template della carta intestata

## Specifiche consigliate

- Formato: A4 (210 × 297 mm)
- Lasciare spazio bianco nella parte centrale per il testo del comunicato
- Margini consigliati: almeno 2cm da ogni lato
- Il testo verrà inserito a partire da circa 8cm dall'alto

## Come caricare il template

1. Sostituisci il file `letterhead.pdf` in questa cartella
2. Riavvia il backend
3. I nuovi comunicati useranno automaticamente il nuovo template

## Formato di esempio

La carta intestata dovrebbe contenere:
- Logo CISL in alto
- Eventuali intestazioni ("CISL Trasporti", indirizzo, contatti)
- Spazio vuoto centrale per il contenuto
- Eventuale piè di pagina

Il testo del comunicato verrà sovrapposto automaticamente sopra il template.
