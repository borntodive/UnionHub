FUNZIONALITÀ

Tutto ciò di cui hai bisogno

Strumenti professionali progettati per piloti e cabin crew, con supporto offline e aggiornamenti in tempo reale.

Calcolatore Busta Paga

Calcola la tua retribuzione netta con regole fiscali italiane aggiornate: IRPEF, detrazioni, contributi INPS. Supporto contratti part-time e storici. Modalità Reverse per calcolare le ore dal guadagno.

FTL Calculator

Calcolo limiti EASA Part-ORO.FTL e Malta Air OMA: massimo FDP, riposo minimo, acclimatazione. Aggiustamenti automatici per turni notturni, disruptive scheduling e zone di acclimatazione.

CTC Calculator

Correzione temperatura fredda (ICAO Doc 9365). Inserisci temperatura aeroporto, quota e altitudini pubblicate — ottieni le correzioni e le altitudini corrette per ogni step. Toggle ft/m e arrotondamento a 100.

Segnalazioni Offline-First

Invia segnalazioni sindacali anche senza connessione. Le segnalazioni in coda si sincronizzano automaticamente al ripristino della rete. Categorie e urgenze sempre disponibili in locale.

Documenti Sindacali

Visualizza comunicati e documenti sindacali direttamente nell'app. PDF viewer integrato con condivisione e download. Generazione PDF con carta intestata FIT e layout bilingue IT/EN.

📲 Come installare l'APK su Android
  1. Abilita le sorgenti sconosciute
    Vai in Impostazioni → Sicurezza (o Impostazioni → App → Accesso speciale → Installa app sconosciute) e abilita il tuo browser o file manager.
  2. Scarica il file APK
    Tocca il pulsante "Scarica Android APK" qui sopra. Il file UnionHub.apk verrà salvato nella cartella Download.
  3. Apri il file scaricato
    Apri il pannello notifiche e tocca la notifica di download, oppure usa un file manager e naviga in Download → UnionHub.apk.
  4. Conferma l'installazione
    Android mostrerà una schermata di avviso. Tocca Installa per procedere. L'app sarà visibile nella home dopo pochi secondi.
  5. Accedi con le tue credenziali
    Usa il crewcode e la password forniti dall'amministrazione per effettuare il primo accesso.
⚠️ Installa solo APK provenienti da fonti ufficiali. Questo file è distribuito direttamente da FIT — Federazione Italiana Trasporti.
📲 How to install the APK on Android
  1. Enable unknown sources
    Go to Settings → Security (or Settings → Apps → Special access → Install unknown apps) and enable your browser or file manager.
  2. Download the APK file
    Tap the "Download Android APK" button above. The file UnionHub.apk will be saved to your Downloads folder.
  3. Open the downloaded file
    Pull down the notification shade and tap the download notification, or use a file manager and navigate to Downloads → UnionHub.apk.
  4. Confirm the installation
    Android will show a warning screen. Tap Install to proceed. The app will appear on your home screen within seconds.
  5. Sign in with your credentials
    Use the crewcode and password provided by your administration for your first login.
⚠️ Only install APKs from official sources. This file is distributed directly by FIT — Federazione Italiana Trasporti.
